== Description ==

Tizra Connect is a simple way to display Tizra-hosted publications in your WordPress site. The list is automatically updated, and displays cover thumbnails for each item. Contents of the list can be managed through Tizra's collection tools. More on Tizra at [http://tizra.com](http://tizra.com).

== Installation ==

1. Upload Tizra Connect to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. A Tizra Connect options page is available for configuration
4. Insert a [tizra_connect] shortcode into your content to display Tizra-hosted publications
